Namespace global

This is the root of the entire anna namespace.

Submodules

  • annaUtil - This is an internal helper library. Do not use it.
  • ast - Internal module for the ast macro.
  • attributes - Internal module for various attribute macros.
  • class - Internal module for the classType macro.
  • collection - Internal module for the __collcetion__ macro.
  • ctime - A simple, low level time API. Unlike most other c* api:s in Anna, this one doesn't closely mimic the underlying C functions.
  • curses - Low level library for dealing with terminal input and output.
  • date - The date module is a high level library for handling dates and time. It currently suffers from the limitation that it can only correctly handle dates within the range of a Unix timestamp. On most 32-bit platforms, this means dates from 1902 to 2038 are correctly handled.
  • enum - Internal module for the enumType macro.
  • error - Simple high level error handling.
  • errorMacros - Internal module for macros related to rasing errors.
  • expandCode - Internal module for the expandCode macro.
  • fiber - A simple cooperative threading library
  • functional - Various tools for functional programming.
  • getText - The Anna getText module is a low level wrapper around the C gettext localization library.
  • global - This is the root of the entire anna namespace.
  • html - HTML tools and utilities
  • i18n - The anna i18n module (i18n is short for internationalization) is a simple high level module for making an Anna program available in different languages.
  • io - The io module is a high level library for handling files. It is currently rather incomplete but contains the basics needed for reading, writing and checking the status of files.
  • iter - Internal module for macros related to iteration.
  • json - A simple JSON library. Only Anna => Json serialization has currently been written. Since Anna has no boolean type, true/false varluse currently aren't handled.
  • lang - The lang module contains the basic language constructs of Anna.
  • listAssign - Internal module for macros related to declaring or assigning to multiple variables in a single expression.
  • macroMisc - Internal module for various default macros.
  • macroUtil - This module contains various macros that are useful when writing macros.
  • mapping - Internal module for the __mapping__ macro.
  • math - Basic mathematical functions and constants
  • monkeypatchContinuation - Internally used module. Don't use directly.
  • monkeypatchLang - Internally used module. Don't use directly.
  • monkeypatchList - Internally used module. Don't use directly.
  • monkeypatchMisc - Internally used module. Don't use directly.
  • monkeypatchNode - Internally used module. Don't use directly.
  • monkeypatchRange - Internally used module. Don't use directly.
  • monkeypatchString - Internally used module. Don't use directly.
  • parser - The parser module contains tools related to compilation of Anna code.
  • propertyAttribute - Internal module for the propertyAttribute macro.
  • range - Internal module for the __range__ macro.
  • readLine - A wrapper for the GNU readline library.
  • reflection - The reflection module contains tools used for run time introspection of functions and types.
  • repl - The anna repl implementation.
  • replTool - Helper commands for the anna repl (interactive prompt) implementation. This module is automatically used by the anna interactive prompt, and there is very rarely any reason to use it manually.
  • struct - Internal module for the struct macro.
  • switch - Internal module for the switch macro.
  • system - The system module contains information and functionality on the currently running system.
  • template - A simple templating library
  • unix - The unix module is the parent module for various low level wrappers for basic Unix functionality.
  • update - Internal module for the macros related to uodating a variable in-place.
