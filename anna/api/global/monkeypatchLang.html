def ImutableList print (String keys:, Object param...) (static);

Print all the unnamed arguments to the function, separated by a space character. Efter the final argument has been printed, a trailing newline is also printed. The arguments are converted to strings by calling their toString method.

The print function understands two named arguments, both of which must be Strings:

  • sep, which is the separator to print between the unnamed arguments (the default is the space character) and
  • end, which is the character to print after the last argument (the default is the newline character).
