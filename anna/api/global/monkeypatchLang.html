  • print - Print all the unnamed arguments to the function.
def ImutableList print (String keys:, Object param...) (static);

Print all the unnamed arguments to the function.

The print function understands two named arguments:

  • sep, which is the separator to print between the unnamed arguments (default is the space character).
  • end, which is the character to print after the last argument (default is the newline character)
