Namespace global.date

The date module is a high level library for handling dates and time. It currently suffers from the limitation that it can only correctly handle dates within the range of a Unix timestamp. On most 32-bit platforms, this means dates from 1902 to 2038 are correctly handled.

Types

  • Date - The Date object represents a point in time with microsecond precision. Dates object are always relative to a specific timezone.
  • Delta - An object representing a time delta, e.g. the difference between two points in time.
  • Timezone - A Timezone object. This object is currently mostly a placeholder, it's functionality is yet to be implemented.
