Namespace global.unix.io

The unix.io module contains low level wrappers for basic unix functionality revolving around input and output.

Unix input/output centers around file descriptors. File descriptors are small, positive integers that can represent open files, directories, pipes or even network sockets.

Submodules

  • fcntlMode - Flags specifying file descriptor manipulations for unix.io.fcntl
  • openMode - Flags determining the mode for unix.io.open
  • seekMode - Different seek modes for use with the unix.io.seek function
  • statMode - Flags used for identifying file status together with a unix.io.Stat object.

Types

  • FLock - File lock information
  • FdSet - A set of file descriptors. Used by unix.io.select.
  • Stat - A structure representing the status of a file.

Constants, variables and properties

  • standardError - File descriptor for error output
  • standardInput - File descriptor for standard input.
  • standardOutput - File descriptor for standard output.
