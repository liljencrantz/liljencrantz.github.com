Namespace global.unix.locale.localeMode

Different locale parts.

Constants, variables and properties

  • all - All of the locale.
  • cType - Regular expression matching, character classification, conversion, case-sensitive comparison, and wide character functions.
  • collate - Regular expression matching (it determines the meaning of range expressions and equivalence classes) and string collation.
  • messages - Localizable natural-language messages.
  • monetary - Monetary formating.
  • numeric - Numeric formating, such as decimal point and thousands separator.
  • time - Date and time formating.
