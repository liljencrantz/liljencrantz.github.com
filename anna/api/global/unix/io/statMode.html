Namespace global.unix.io.statMode

Flags used for identifying file status together with a unix.io.Stat object.

Constants, variables and properties

  • block - Block device.
  • character - Character device.
  • directory - Directory.
  • fifo - FIFO.
  • groupAll - Mask for all group permissions.
  • groupExecute - Group has execute permission.
  • groupRead - Group has read permission.
  • groupwrite - Group has write permission.
  • link - Symbolic link.
  • otherAll - Mask for permissions for others (not in group).
  • otherExecute - Others have execute permission.
  • otherRead - Others have read permission.
  • otherwrite - Others have write permission.
  • regular - Regular file
  • sgid - Set-group-ID bit.
  • socket - Socket
  • sticky - Sticky bit.
  • suid - Set UID bit.
  • userAll - Mask for all file owner permissions.
  • userExecute - File owner execute permission.
  • userRead - File owner read permission.
  • userwrite - File owner write permission.
