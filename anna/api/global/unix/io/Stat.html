Type global.unix.io.Stat

A structure representing the status of a file.

This is a straight conversion of the C struct stat.

Constants, variables and properties

  • __type__ - The type of this object
  • atime - Time of latest acccess.
  • blksize - Block size for file system IO.
  • blocks - Number of 512 byte blocks allocated.
  • ctime - Time of latest change.
  • dev - ID of device containing file
  • gid - Group ID of owner.
  • ino - Inode number
  • mode - File protection mask
  • mtime - Time of latest modification.
  • nlink - Number of hard links.
  • rdev - Device ID.
  • size - Total size in bytes.
  • uid - User ID of owner.
