Namespace global.unix.env

The unix.env module contains low level wrappers for basic unix functionality revolving around environment variables.

Methods and functions

  • clearenv - Removes all environemnt variables. Equivalanet to the C clearenv function.
  • getenv - Return the current value of the given environment variable. Equivalanet to the C getenv function.
  • setenv - Assign a new value to the environment variable with the given name. Equivalanet to the C setenv function.
  • unsetenv - Delete the specified environment variable. Equivalanet to the C unsetenv function.
