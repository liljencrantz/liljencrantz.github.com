Namespace global.unix.proc

The unix.proc module contains low level wrappers for basic unix functionality revolving around processes and signals.

Submodules

  • signal - All known signals
  • signalfdFlag - Flags determining the mode for unix.proc.signalfd
  • sigprocmaskFlag - Flags determining the mode for unix.iosigprocmask

Types

  • SignalInfoFd - A structure describing a recieved signal.
  • SignalSet - A set of signals. Used by the unix.proc.signalfd and unix.proc.sigprocmask functions.

Methods and functions

  • exec - Execute the specified program.
  • exit - Terminate the calling process.
  • fork - Create a child process.
  • getpid - Get process ID.
  • getppid - Get parent process ID.
  • getsid - Get session ID.
  • kill - Send a signal to a process.
  • readSignal - Read one signal information structure form the specified file descriptor.
  • setsid - Set session ID.
  • signalfd - Create a file descriptor for accepting signals.
  • sigprocmask - Examine and change what signals are currently blocked by this process.
  • wait - Wait for a process to change state.
  • waitpid - Wait for a process to change state.
