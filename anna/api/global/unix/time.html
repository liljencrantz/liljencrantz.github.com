Namespace global.unix.time

The unix.time module contains low level wrappers for basic unix functionality revolving around timekeeping.

Types

  • TimeVal - A data structure representing a point in time as the amount of time that has elapsed since the epoch.
  • TimeZone - A deprecated data structure that used to be used for representing a time zone.

Methods and functions

  • gettimeofday - Gets the current system time. Equivalanet to the C gettimeofday function.
