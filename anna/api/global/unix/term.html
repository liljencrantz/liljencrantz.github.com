Namespace global.unix.term

The unix.term module contains low level wrappers for basic unix functionality revolving around terminal handling.

Submodules

  • action - The different action modes used by unix.term.setAttr.
  • flag - The different modes of a Termios structure.

Types

  • Termios - A structure representing the state of a terminal.

Methods and functions

  • getAttr - Get the current state of a terminal.
  • setAttr - Set the current state of the terminal.
