Namespace global.unix.rLimit

The unix.rLimit module contains low level wrappers for basic unix functionality revolving around system resource limits.

Submodules

  • rLimitMode - The different limits that can be queried with getRLimit and stRLimit.

Types

  • RLimit - A structure representing the current and maximum value of a system resource usage limit.

Methods and functions

  • getRLimit - Get resource limit.
  • setRLimit - Set resource limit.
