Namespace global.unix.user

The unix.user module contains low level wrappers for basic unix functionality revolving around users and groups.

Methods and functions

  • getegid - Get the effective group ID of the calling process.
  • geteuid - Get the effective user ID of the calling process.
  • getgid - Get the real group ID of the calling process.
  • getgroups - Get list of supplamentary group IDs.
  • getpgid - Get process group.
  • getuid - Get the real user ID of the calling process.
  • setegid - Set the effective group ID of the calling process.
  • seteuid - Set the effective user ID of the calling process.
  • setgid - Set the effective group ID of the calling process. If the calling user is root, the real group ID is also set.
  • setgroups - Set list of supplamentary group IDs.
  • setpgid - Set process group.
  • setuid - Set the effective user ID of the calling process. If the calling user is root, the real user ID is also set.
