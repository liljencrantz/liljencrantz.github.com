  • ast
def Node ast (Call node);
A macro that takes an AST tree and returns it verbatim. This is useful for creating code templates, and often used togerther with the AST Node formating function. Example: ast(%value := 0) % ["value": Identifier(?, "someVariableName") ]; The above example will return an AST that is equivalent to the following code: someVariableName := 0;
