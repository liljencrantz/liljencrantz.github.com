Namespace global.lang.Char

The Char type is the basic character type of the Anna language.

Anna Char objects are implemented using wide characters, e.g. the wchar_t type in C. This means that every character takes the same amount of memory. Nearly all Char values, (those with 30 or fewer bits used), are stored directly on the stack and use no heap memory at all.

Anna Char objects are imutable, meaning their value never changes.
