Type global.lang.HashMap

A HashMap represents a set of mappings from keys to values.

Internally, the HashMap type is implemented as a Hash table. Every key must implement the equality comapison method and the hashCode method.

Setting new mappings and getting the corresponding value for a specified key both have an O(1) amortized time, provided that the keys correctly implement the hashCode method.
