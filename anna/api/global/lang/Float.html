Type global.lang.Float

The float Type represents a double precision floating point number.

Float objects are imutable, can be used as hash keys and implement the basic arithmetic operations like addition, multiplication, etc. Arithmetic operations on combinations of Floats and Ints result in a Float object. Arithmetic operations on Floats ad Complex numbers result in a Complex object.
