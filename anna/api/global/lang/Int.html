Namespace global.lang.Int

The Int type is the basic integer type of the Anna language.

Anna Int objects are arbitrary precision, i.e. they never overflow. Small integer values, numbers that use 29 bits or less to represent (excluding the sign bit, so 30 bits total of data) can be stored directly on the stack and use no heap memory at all. Larger integers are implemented using a multiple precision library such as GNU MP.

Anna Int objects are imutable, meaning their value never changes.
