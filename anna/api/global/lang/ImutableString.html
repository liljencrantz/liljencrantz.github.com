Namespace global.lang.ImutableString

The ImutableString type is type representing an imutable (unchanging) character string.

In order to obtain a mutable (changable) version of an ImutableString, use the thaw property. On MutableString, this property is a returns the original string.

String literals, such as "test" are all instances of ImutableString. In order to obtain a mutable (changable) version of an ImutableString, use the thaw property.
