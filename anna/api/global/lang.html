Namespace global.lang

The lang module contains the basic language constructs of Anna.

Basic types like numbers, character strings and data buffers live in the lang module, as do a few nearly universally used functions, like print.

Types

  • Buffer - The buffer type represents a byte oriented mutable array of binary data. It is primarily used for reading and writing from file descriptors, but can also be used to encode text strings to a specific encoding and various other low level tasks.
  • Char - The Char type is the basic character type of the Anna language.
  • Complex - The Complex type is the basic complex floating point number type of the Anna language.
  • Float - The float Type represents a double precision floating point number.
  • HashMap - A HashMap represents a set of mappings from keys to values.
  • ImutableList - The ImutableList type is type representing an imutable (unchanging) list of objects.
  • ImutableString - The ImutableString type is type representing an imutable (unchanging) character string.
  • Int - The Int type is the basic integer type of the Anna language.
  • List - The List type represents any list, either a mutable or imutable one. It is the intersection of the MutableList and the ImutableList.
  • MutableList - The MutableList type is type representing a mutable (changing) list of objects.
  • MutableString - The MutableString type is type representing a mutable (changing) character string.
  • Null - The Null type is the type of the null object.
  • Object - The Object type is the base type of all other types in Anna.
  • Pair - A Pair represents a mapping from a key to a value.
  • Range - A Range represents a sequence of equally spaced integers.
  • String - The String type represents any character string, either a mutable or imutable one. It is the intersection of the MutableString and the ImutableString.

Operators

  • ! (Boolean negation) __not__

Methods and functions

  • __not__ - Negates the value. Returns 1 of the input is null, null otherwise.
  • callCC - Call with current continuation.
  • nothing - The nothing function returns the last argument in the parameter list (Or null if given no arguments). This function is sometimes useful when performing meta-programming and wishing to do multiple things in a single expression.
  • print - Print all the unnamed arguments to the function.
  • printInternal - Print the specified String to standard output. This is a non-standard internal helper function, do not use it directly. Use the print function instead.

def ImutableList print (String keys:, Object param...) (static);

Print all the unnamed arguments to the function.

The print function understands two named arguments:

  • sep, which is the separator to print between the unnamed arguments (default is the space character).
  • end, which is the character to print after the last argument (default is the newline character)
