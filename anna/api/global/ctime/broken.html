Namespace global.ctime.broken

The constants found in this module are used to access the different fields of a broken down time as returned by ctime.breakTime.

Constants, variables and properties

  • dayOfMonth - Number of days since first day of month.
  • dayOfWeek - Day of week, Sunday is 0.
  • dayOfYear - Number of days since January 1:st.
  • daylightSaving? - If non-null, daylight savings time is in effect
  • hour - Number of hours past midnight.
  • minute - Number of minutes past last whole hour.
  • month - Month, January is 0.
  • second - Number of seconds past last whole minute.
  • year - Number of years since 1900.
