Namespace global.math

Basic mathematical functions and constants

Constants, variables and properties

  • e - The mathematical constant e, i.e. the base of the natural logarithm.
  • pi - The mathematical constant pi, i.e. the circumference of a circle divided by its diameter.

Methods and functions

  • acos - Computes the arc cosine of value, where value is given in radians.
  • asin - Computes the arc sine of value, where value is given in radians.
  • atan - Computes the arc tangent of value, where value is given in radians.
  • cos - Computes the cosine of value, where value is given in radians.
  • log - Computes the natural logarithm of value.
  • log10 - Computes the base 10 logarithm of value.
  • log2 - Computes the base 2 logarithm of value.
  • sin - Computes the sine of value, where value is given in radians.
  • sqrt - Computes the square root of value.
  • tan - Computes the tangent of value, where value is given in radians.
