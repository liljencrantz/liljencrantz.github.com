Namespace global.json

A simple JSON library. Only Anna => Json serialization has currently been implemented.

Methods and functions

  • to - Serialize the specified object into JSON. Since Anna has no boolean type, true/false values currently aren't handled.
def String to (Object o) (static);

Serialize the specified object into JSON. Since Anna has no boolean type, true/false values currently aren't handled.

Usage:

json.to(["key": [1,2,?]}])
// Result: "{"key": [1, 2, null]}"
