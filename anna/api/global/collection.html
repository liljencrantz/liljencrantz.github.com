Namespace global.collection

Various functions that can be used to transform collections such as Lists and HashMap.

Methods and functions

  • flatten - Return a MutableList containing all the elements of the List of Lists given as input.
  • zip - Return a MutableList of ImutableList objects, where each ImutableList contains the i-th element from each of the argument lists. The returned list is truncated in length to the length of the shortest argument sequence. The specialization of the ImutableList is the intersection of the static types of all the argument lists.
