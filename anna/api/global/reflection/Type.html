Type global.reflection.Type

The Type type represents an Anna Type. It is mostly used for object introspection.

Types that this module can mask as

global.lang.Object, global.markdown.PreProcessor, global.date.Timezone

Constants, variables and properties

  • __type__ - The type of this object
  • attribute - All attributes specified for this type.
  • filename - The name of the file in which this type was defined, if any.
  • member - A list of all members of this type.
  • module? - Is true if this type represents a module.
  • name - The name of this type.
