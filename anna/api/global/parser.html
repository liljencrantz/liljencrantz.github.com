Namespace global.parser

The parser module contains tools related to compilation of Anna code.

The basic types used when representing the source code as an AST tree, as well as tools for parsing and compiling a piece of code into a module live in this module.

Methods and functions

  • compile - Send the specified AST to the compiler and return the resulting code. The AST must specify a valid function definition. Warning: Error recovery in the compiler is currently rudementary. Any compilation errors are liable to put the entire interpreter into an invalid state, resulting in a forced exit.
  • identifier - Generate a unique identifier name, suitable for use as an internal identifier in autogenerated code
  • parse - Parse the specified string and return an AST tree representing it
